Let's get to work

You will have been invited to Nifty, the all-in-one project management tool we use to set tasks, communicate, share files, laugh until the sun comes up. It can be a bit overwhelming at first but the most important things at this point are Tasks and Discussions.

Tasks

As you've rightly assumed, tasks is where you'll add tasks to be completed. Create as many tasks as you'd like as often as you'd like. The smaller the better so we can ship sooner. Only one task can be done at any one time (unless on the Express plan) so bear this in mind.

Create tasks using the big "create" button at the top right and use the FlowSt8 Template which will prompt you for all the important information but please include;

  • Task title - what the success state of the task is (e.g. Fix signup form)
  • Task description - any additional information relating to the task
  • Figma files
  • Logins, assets, anything additional
  • Tags and due date if necessary
  • Bonus points: Subtasks in order of completion

Important: Please give me access to your Webflow account and any other external accounts that you think i'll need access to. My Webflow account username is @flowst8.
